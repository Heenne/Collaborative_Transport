Detailed Description

template<class T>
class MsgOrientationFeedForward< T >

A Ros implementation of the OrientationFeeedForward class. It contains a ros Subscriber that listens to a specific message and uses the message data as input data for the feed forward control.

Template Parameters
TMessage type the subcriber is listeneing to. A convertMsg fucntion has to be implemented for it.
